        # Privacy Awareness - Demo Project

This is a simple React + Vite demo project prepared for uploading to GitHub and deploying to Vercel/Netlify.

## How to run locally

1. Install dependencies:

```
npm install
```

2. Start dev server:

```
npm run dev
```

Open the URL shown (usually http://localhost:5173/)

## If you want to deploy
Upload this repository to GitHub, then import it into Vercel (Login with GitHub) and `Deploy`.
